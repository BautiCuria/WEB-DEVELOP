Tecnologias basicas del desarrollo web

En esta pagina listaremos de manera breve las tres tecnologias de la web.

¿Cuales son estas tecnologias?

HTML

Es un lenguaje de marcado donde marcamos que elementos existen dentro de una pagina utilizando etiquetas.

CSS

Es un lenguaje de estilos que nos permite elegir como se ven y donde se ven los elementos de nuestra pagina.

JavasCript

Es un lenguaje de programacion en el cual escribimos la interactividad de nuestro sitio web.

¿Por que querriamos aprender estas tecnologias?

Aprender desarrollo web nos beneficiara de distintas maneras:

  • Podremos buscar trabajo en un rubro que no para de crecer.
  • Nos habilita a trabajar para empresas de todo el mundo o aser independientes.
  • Es un conjunto de tecnologias no muy complicadas que nos intraducen al mundo de la programacion.
